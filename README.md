# titanic_survival_classifier

• Deployed 4 survival prediction models on a train-test set of 889 Titanic passengers, using 6 demographic/ticket features

• Identified intuitive decision patterns based on gender, class, and fare price, leading to a 45% reduction in prediction errors

• Benchmarked LDA, QDA, classification trees, and logistic regression using supervised learning methodologies, ultimately
selecting a model that balanced interpretability with strong predictive accuracy for deployment
